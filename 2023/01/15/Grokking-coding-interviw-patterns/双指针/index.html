双指针

如何想起使用双指针：find two data elements in an array that satisfy a certain condition

可以使用双指针的例子

  • 查看string是不是回文串
  • 反转数组
  • 在排好序的array中寻找一组pair等于target

什么样的情况下问题符合双指针使用条件

需要同时满足两个条件

  1. 当遍历的输入数据可以线性方式遍历（数组，链表，字符串）
  2. 输入数据是sort好的或者按照问题相关的方式排序的

当其中一个条件满足时就不可以用

  1. 输入数据不可线性方式遍历
  2. 问题需要对答案空间进行exhaustive search（也就是消除一个答案不会消除其他答案）

实际应用场景

产品建议： 当客户购买的产品没有达到免运费标准时，双指针可以用来添加pair物品达到免运费。

判断单个单词是不是回文串

Naive想法

反转字符串然后和原数组比较

双指针

时间复杂度：O(n)n是string里面的字符数量
空间复杂度：O(1)因为用了constant space储存了两个indices

Sum of Three Values(三数之和)

Naive想法

三个 for loop
time complexity O(n3)
space complexity O(1)没用额外空间

双指针

  1. sort array(sort 完以后才可以进行双指针运算)
  2. loop整个array对于每个iteration创建双指针
  3. low = i + 1, high = len(nums) - 1
  4. 计算当前当前遍历的元素和low和high对应的元素的和
  5. 如果大于就移动右边high，小于移动左边low.等于直接返回
  6. 否则return False

Time complexity O(Nlogn) + O(n2)
Space complexity sort在 O(logn)～O(n)之间。对于python是O(n)

Reverse Words in a String反转字符串中的单词

Given a sentence, reverse the order of its words without affecting the order of letters within a given word.

部分正则表达式用法（re）

双指针

  1. reverse the total string(注意string时immutable所有需要转换成list)
  2. reverse each word

Time complexity O(n + n) = O(n) 因为遍历了两次string
Space complexity O(n) 因为python string是immutable所以我们copy成了一个list类型

Valid Palindrome II 有效回文串2

Leetcode链接

双指针

  1. 头尾双指针
  2. 如果相同就一直走
  3. 如果遇到不相同的就skip其中的一个元素判断子串是否相同。skip另一个指针然后判断是否相同
  4. 如果其中的一个子串可以构成回文串说明可以否则不行

Time Complexity O(n) 遍历整个字符串。遇到不同字符时判断子串也是O(n)
Space Complexity O(1)常量空间
